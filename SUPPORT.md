# Support

Tungsten is a personal research project. Support is best-effort.

## Getting Help

- **Bug reports:** [Open an issue](https://github.com/Spinning-Cat-Studios/tungsten/issues) with a minimal `.tg` reproduction
- **Questions & discussion:** [GitHub Discussions](https://github.com/Spinning-Cat-Studios/tungsten/discussions)
- **Security issues:** See [SECURITY.md](SECURITY.md)

## What to Include in a Bug Report

- Tungsten version (`tungsten --version`)
- Operating system and architecture
- Minimal `.tg` file that reproduces the issue
- Expected vs actual behaviour
- Full error output

## What This Project Does Not Offer

- Guaranteed response times
- Production support or SLAs
- Feature development on request (see [CONTRIBUTING.md](CONTRIBUTING.md))
